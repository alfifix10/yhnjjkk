# Jeerani — دردشة جيراني

## ما هو المشروع
تطبيق ويب للدردشة المجهولة المؤقتة المبنية على الموقع الجغرافي. المستخدم يدخل باسم مستعار، يرى الأشخاص القريبين، ويتحدث معهم 1-على-1. لا تسجيل، لا حسابات، الرسائل تُحذف عند الخروج.

**الموقع:** jeerani.net  
**الاستضافة:** GitHub Pages + Cloudflare DNS  
**قاعدة البيانات:** Firebase Realtime Database

## الملفات الرئيسية
- `index.html` — الصفحة الرئيسية (شاشة الدخول + الجيران + الدردشة)
- `js/app.js` — المنطق الكامل (GPS، Firebase، دردشة، حظر، إشعارات)
- `css/style.css` — التصميم (داكن، RTL، متجاوب)
- `admin.html` — لوحة التحكم (مستخدمين، محادثات، محظورين)
- `firebase-rules.json` — قواعد Firebase (مرجع — تُحدَّث يدوياً في Console)
- `firebase-messaging-sw.js` — Service Worker للإشعارات
- `manifest.json` — إعدادات PWA
- `CNAME` — النطاق jeerani.net

## Firebase nodes
- `online/` — المتصلين حالياً (يُحذف عند الخروج بـ onDisconnect)
- `msgs/{uid}` — الرسائل المباشرة (تُحذف بعد الاستلام)
- `logs/` — سجل كل الرسائل (للأدمن)
- `typing/{uid1}/{uid2}` — مؤشر الكتابة + إشعارات القراءة
- `banned/{id}` — المحظورين بالـ ID + البصمة

## الثوابت المهمة (في app.js)
- `MAX_NEARBY_KM = 100` — أقصى مسافة لعرض الجيران
- `GPS_MIN_ACCURACY = 200` — أقل دقة GPS مقبولة (متر)
- `GPS_MAX_JUMP_KM = 50` — كشف التلاعب بالموقع
- `MSG_COOLDOWN_MS = 500` — الحد الأدنى بين الرسائل
- `MSG_PER_MINUTE = 20` — حد الرسائل بالدقيقة
- `DAILY_MSG_LIMIT = 200` — حد الرسائل اليومي
- `HEARTBEAT_MS = 60000` — فترة تحديث الحضور

## لوحة التحكم (admin.html)
- **كلمة المرور:** Mm123 (مشفرة بهاش في الكود)
- **3 أقسام:** المستخدمين، المحادثات، المحظورين
- المستخدمين يُستخرجون من `logs` + `online`، ويُحفظون في localStorage
- المحظور يُحظر بالـ ID + البصمة (يمنع التحايل بحذف بيانات المتصفح)
- المحظور يختفي من قائمة المستخدمين ويظهر فقط في المحظورين

## نظام الحظر
- فحص الحظر عند الدخول اليدوي (زر ادخل) بالـ ID + البصمة
- مراقبة مباشرة أثناء الاستخدام — المحظور يُطرد فوراً
- البصمة = canvas + شاشة + متصفح + timezone (ثابتة حتى لو مسح البيانات)
- `checkBanned()` فيها timeout 3 ثواني حتى لا يعلّق الزر

## نظام GPS
- `watchPosition` مع تحسين تدريجي للدقة
- IP geolocation كـ fallback لو GPS مرفوض
- قفزات أكبر من 50 كم تُرفض (كشف تلاعب)
- الإحداثيات تُقرّب لـ 3 خانات عشرية (~100 متر) للخصوصية

## الإشعارات
- 🔊 زر السماعة — كتم/تفعيل صوت الرسائل
- 🔔 زر الجرس — تنبيه عند دخول جار قريب (صوت + اهتزاز + إشعار متصفح)
- الجرس يعمل فقط والتطبيق مفتوح (Push Notifications تحتاج FCM + Cloud Functions)

## تعليمات التطوير
- اللغة: عربي RTL
- التصميم: داكن فاخر
- لا تضف ملفات جديدة إلا للضرورة
- لا تعدل القواعد في firebase-rules.json بدون إعلام المستخدم (تحتاج تحديث يدوي في Console)
- عند الـ commit اكتب رسالة واضحة بالإنجليزية
- ادفع دائماً لـ `main` (GitHub Pages)
