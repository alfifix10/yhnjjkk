# قواعد جيرانك — الخطوط العريضة

## 1. البساطة أولاً
- شاشة واحدة = وظيفة واحدة (دخول → قائمة → دردشة)
- لا تسجيل، لا إيميل، لا رقم جوال — اسم مستعار فقط
- لا إعدادات معقدة — كل شي يشتغل تلقائياً

## 2. الهوية
- كل جهاز = هوية واحدة ثابتة (بصمة + ID)
- الاسم يتغيّر، الهوية لا
- الحظر دائم — ما فيه تحايل

## 3. الرسائل
- نص فقط (لا صور، لا ملفات، لا صوت)
- حد 500 حرف لكل رسالة
- حد 20 رسالة بالدقيقة
- 500ms بين كل رسالة
- الرسائل تنحفظ محلياً (آخر 100 لكل محادثة)

## 4. الخصوصية
- الموقع الجغرافي اختياري
- الإحداثيات تتقرّب لـ 100 متر
- لا بيانات شخصية تُخزّن على السيرفر
- المحادثات تنحفظ على جهازك فقط — مو على السيرفر

## 5. الأمان
- فلتر كلمات بذيئة (مستقبلاً)
- حظر المستخدمين المزعجين
- لا روابط خارجية مشبوهة (مستقبلاً)
- Firebase Security Rules تمنع التلاعب بالقاعدة

## 6. التصميم
- داكن دائماً (Dark Mode)
- RTL عربي بالكامل
- يشتغل على الجوال والكمبيوتر بنفس الشكل
- لا إطارات معقدة — بسيط ونظيف
- الأنيميشن خفيف — لا يبطئ الجهاز

## 7. الأداء
- ما فيه تحميل صفحات — كل شي SPA (صفحة واحدة)
- Firebase Realtime — لا polling ولا refresh
- الكود خفيف — ملف JS واحد + CSS واحد
- يشتغل حتى بنت ضعيف

## 8. قواعد لا نكسرها
- لا نضيف ميزة تعقّد التجربة
- لا نضيف ميزة بدون ما نختبر كل سيناريوهاتها
- لو الميزة تحتاج شرح = تصميمها غلط
- الأقل أفضل — كل شي زائد يُحذف
